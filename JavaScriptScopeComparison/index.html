About the project

I made this project because I was curious about the differences between Var, Let, and Const variables while I was revising my JavaScript Fundamentals

Begin by clicking on either Global Scope or Function Scope buttons.
  • The first row represents the scope of the variables and where they were declared
  • In the second row I will attempt to reassign the variables
  • In the last row I will try to call the variables from another function. If the call is successful, I will attempt to reassign the variables again
